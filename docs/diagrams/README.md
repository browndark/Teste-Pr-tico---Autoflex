# Diagrams - Architecture & Design Documentation

This folder contains visual diagrams and renderings for the Inventory Management System architecture, design, and algorithms.

## Directory Structure

```
diagrams/
├── architecture/          # System architecture diagrams
├── database/             # Database (ER diagrams)
├── use-case/             # Use case diagrams
├── sequence/             # Sequence and flow diagrams
├── deployment/           # Deployment architecture
├── algorithm/            # Algorithm flowcharts
└── README.md             # This file
```

## Diagram Categories

### Architecture
System component relationships, layers, and overall design.

**Storage:** `architecture/`

### Database
Entity relationship diagrams, table structures, and relationships.

**Storage:** `database/`

### Use Cases
User interactions and system capabilities.

**Storage:** `use-case/`

### Sequence Diagrams
Step-by-step flows and API interactions.

**Storage:** `sequence/`

### Deployment
Production environment setup and infrastructure.

**Storage:** `deployment/`

### Algorithm
Algorithm logic flows and decision trees.

**Storage:** `algorithm/`

---

## File Organization

Each subdirectory contains:
- PNG images (rendered diagrams)
- SVG files (vector diagrams for scaling)
- PNG variants (different sizes/resolutions)

## Usage

Include diagrams in documentation:

```markdown
![System Architecture](docs/diagrams/architecture/system_architecture.png)
```

Or reference in presentations:

```html
<img src="docs/diagrams/database/entity_relationship.svg" />
```
